4. Functions / Methods in Java

In Java, a method is a block of code written inside a class, which performs a specific task. Every method must belong to a class (i.e. you canâ€™t have stand-alone functions). Methods let you reuse logic, structure your code, and make it more readable.

Parts of a Method

  • Modifier / Access Specifier: e.g. public, private, etc.
  • Return Type: the type of value the method returns, or void if it returns nothing.
  • Method Name: follows naming rules (lowercase, camelCase, etc.).
  • Parameters: optional inputs the method accepts (e.g. (int a, int b)).
  • Method Body: the statements inside { â€¦ } that specify what the method does.

// Example: defining & calling a method

    public static int add(int a, int b) {

        return sum;

    public static void main(String[] args) {
        int result = add(5, 7);  // calling the method
        System.out.println("Sum = " + result);
    }
