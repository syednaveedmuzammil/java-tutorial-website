Control Statements

Control flow refers to the order in which statements within a program execute. While programs typically follow a sequential flow from top to bottom, there are scenarios where we need more flexibility. This article provides a clear understanding about everything you need to know about Control Flow Statements.

Types of Control Statements


1. Selection Statements

Selection statements allow a program to make decisions and execute certain parts of code based on conditions. They help control the flow of execution.

  • if statement: Executes a block of code only if a specified condition is true.
  • if-else statement: Executes one block of code if the condition is true, and another block if itâ€™s false.
  • nested if statement: An if statement inside another if, used for multiple conditions.
  • switch statement: Allows you to choose between multiple options based on the value of a variable.

// Example: if-else statement
int num = 10;

if(num > 0) {
    System.out.println("Positive number");   // Executes if the condition is true
} else {
    System.out.println("Non-positive number"); // Executes if the condition is false
}

2. Iteration Statements (Loops)

Iteration statements (also called loops) are used to execute a block of code repeatedly as long as a given condition is true.

  • for loop: Used when you know in advance how many times you want to execute a block of code.
  • while loop: Executes a block of code repeatedly as long as the condition is true.
  • do-while loop: Similar to while loop, but executes the block at least once before checking the condition.
  • for-each loop: Used to iterate over arrays or collections easily.

// Example: for loop
for(int i = 1; i <= 5; i++) {
    System.out.println("Java Loop: " + i); // Prints 1 to 5
}

ðŸ”¹ Here, the for loop runs 5 times. The variable i starts from 1 and increases until it reaches 5, printing each value.

3. Jump Statements

Jump statements are used to transfer control to another part of the program. They help manage how and when loops or methods terminate or skip iterations.

  • break: Immediately exits from the current loop or switch statement.
  • continue: Skips the current iteration and moves to the next iteration of the loop.
  • return: Exits from the current method and optionally returns a value.

// Example: break and continue
for(int i = 1; i <= 5; i++) {
    if(i == 3)
        continue; // skips iteration when i = 3
    if(i == 5)
        break; // exits the loop when i = 5
    System.out.println(i);
}
