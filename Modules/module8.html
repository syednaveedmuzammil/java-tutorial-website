Arrays in Java

An array is a collection of elements of the same data type stored in a continuous memory location. Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

Declaring and Initializing Arrays

The size of an array must be defined when it is created. You can either specify the size during declaration or directly initialize it with values, which automatically determines its size. 


// Example: Declaring and initializing arrays
int[] numbers = new int[5];       // Declaring array with size
int[] values = {10, 20, 30, 40};  // Initialization with values

// Accessing array elements
System.out.println(values[0]); // Output: 10

// Using a loop to print all elements
for (int i = 0; i < values.length; i++) {
    System.out.println(values[i]);
}

Types of Arrays

  • Single-Dimensional Array: A simple linear array that stores elements in one row.
  • Multi-Dimensional Array: An array of arrays, such as a 2D matrix.

// Example: Two-Dimensional Array
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing 2D array elements
System.out.println(matrix[1][2]); // Output: 6

Iterating Over Arrays

You can also use an enhanced for loop (for-each loop) to traverse arrays easily. 


// Example: Enhanced for loop
int[] marks = {85, 90, 75, 95};
for (int mark : marks) {
    System.out.println(mark);
}

Arrays in Java are fixed in size and can hold both primitive data types and objects. Once created, their length cannot be changed.

7. Strings in Java

A string is a sequence of characters enclosed within double quotes. In Java, strings are objects of the String class located in the java.lang package.

Creating Strings

Strings can be created in two main ways: using string literals or using the new keyword.


// Example: Creating Strings
String s1 = "Hello";
String s2 = new String("Java");

System.out.println(s1); // Output: Hello
System.out.println(s2); // Output: Java

Common String Methods

Java provides several built-in methods to work with strings.

  • length() — returns the number of characters in the string.
  • charAt(int index) — returns the character at a specific index.
  • substring() — extracts a part of the string.
  • toLowerCase(), toUpperCase() — converts the string to lower or upper case.
  • equals() — compares two strings for equality.
  • concat() — joins two strings.

// Example: Using String methods
String text = "Java Programming";

System.out.println(text.length());        // Output: 16
System.out.println(text.charAt(5));       // Output: P
System.out.println(text.substring(5, 16)); // Output: Programming
System.out.println(text.toUpperCase());   // Output: JAVA PROGRAMMING

String Comparison

Strings can be compared using equals() for content comparison and == for reference comparison. 


// Example: String comparison
String a = "Java";
String b = "Java";
String c = new String("Java");

System.out.println(a == b);      // true (same memory reference)
System.out.println(a == c);      // false (different object)
System.out.println(a.equals(c)); // true (same content)

Strings in Java are immutable, meaning once created, their value cannot be changed. Any modification creates a new String object.
