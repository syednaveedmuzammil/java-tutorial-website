2. Data Types in Java

In Java, data types specify the different sizes and values that can be stored in a variable. They are essential for defining the kind of data a variable can hold, such as numbers, characters, or text.

1. Types of Data Types

Java has two main categories of data types:

  • Primitive Data Types — predefined by Java and named by a keyword.
  • Non-Primitive Data Types — also known as reference types, created by the programmer.

2. Primitive Data Types

There are 8 primitive data types in Java:

Data Type Size Default Value Description
byte1 byte0Stores small integer values (-128 to 127)
short2 bytes0Stores larger integers (-32,768 to 32,767)
int4 bytes0Commonly used for whole numbers
long8 bytes0LUsed for very large integer values
float4 bytes0.0fUsed for decimal (floating-point) numbers
double8 bytes0.0dUsed for precise decimal values
char2 bytes'\\u0000'Stores a single character
boolean1 bitfalseStores true or false values

Example: Primitive Data Types


// Example demonstrating primitive data types
public class DataTypesExample {
    public static void main(String[] args) {
        int age = 25;
        float height = 5.9f;
        char grade = 'A';
        boolean isStudent = true;

        System.out.println("Age: " + age);
        System.out.println("Height: " + height);
        System.out.println("Grade: " + grade);
        System.out.println("Is Student: " + isStudent);
    }
}

3. Non-Primitive Data Types

Non-primitive data types are objects and can be used to call methods. They include classes, arrays, strings, and interfaces. 


// Example of non-primitive data types
public class NonPrimitiveExample {
    public static void main(String[] args) {
        String name = "Java Programming";
        int[] numbers = {10, 20, 30};
        System.out.println("Name: " + name);
        System.out.println("First number: " + numbers[0]);
    }
}

Java is a statically typed language — every variable must be declared with a specific data type before it is used.
