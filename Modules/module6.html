Type Casting in Java

Type Casting in Java is the process of converting one data type into another. This conversion is useful when you need to store or operate on values of different types in a single expression or computation.

There are two main types of type casting in Java:

  • Widening (Implicit) Casting ‚Äì automatic conversion performed by the compiler.
  • Narrowing (Explicit) Casting ‚Äì manual conversion performed by the programmer.

1. Widening (Implicit) Type Casting

Widening conversion happens automatically when a smaller data type is converted into a larger data type. There is no data loss, and it is also known as automatic type conversion. 


// Example 1: Integer to Double
int num = 25;
double result = num; // Automatic conversion
System.out.println(result); // Output: 25.0
      

// Example 2: Char to Int
char letter = 'A';
int ascii = letter; // Automatically converts 'A' to its ASCII value
System.out.println(ascii); // Output: 65

Widening conversion is safe because Java ensures that no information is lost when converting from a smaller to a larger data type.

2. Narrowing (Explicit) Type Casting

Narrowing conversion happens when a larger data type is converted into a smaller data type. This type of conversion must be done manually using parentheses, and there is a possibility of data loss or precision loss. 


// Example 1: Double to Int
double price = 99.99;
int finalPrice = (int) price; // Manual conversion
System.out.println(finalPrice); // Output: 99 (fractional part lost)
      

// Example 2: Long to Short
long bigNum = 32000;
short smallNum = (short) bigNum; // Manual conversion
System.out.println(smallNum); // Output: 32000 (same, fits within short range)
      

// Example 3: Overflow during narrowing
long overflowNum = 130000;
short converted = (short) overflowNum; 
System.out.println(converted); // Output: -31072 (data overflow)

3. Mixed-Type Expressions

Java automatically promotes smaller types to larger types in mathematical expressions. This is part of type promotion rules in Java. 


// Example
int a = 10;
float b = 5.5f;
double result = a + b; // int promoted to float, then to double
System.out.println(result); // Output: 15.5

In the above example, a (int) is automatically converted to a float before addition, and the result is stored as a double.

4. Real-World Example

Type casting is widely used when dealing with user input, files, or APIs that return generic data. For example, reading a numeric value from a file or web form often comes as a String, and you must cast it into an int or double for mathematical operations. 


// Example: Converting String to int
String input = "123";
int number = Integer.parseInt(input);
System.out.println(number + 10); // Output: 133

You can also use casting in classes and inheritance when one object type is converted into another type in the same hierarchy ‚Äî this is called reference casting. 


// Example: Upcasting and Downcasting
class Animal {
    void sound() { System.out.println("Animal sound"); }
}

class Dog extends Animal {
    void sound() { System.out.println("Bark"); }
    void wagTail() { System.out.println("Wagging tail"); }
}

public class Main {
    public static void main(String[] args) {
        Animal a = new Dog();  // Upcasting (safe)
        a.sound();             // Output: Bark

        Dog d = (Dog) a;       // Downcasting
        d.wagTail();           // Output: Wagging tail
    }
}
