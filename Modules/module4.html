Java Operators

In Java, operators are special symbols used to perform specific operations on one, two, or more operands. They help manipulate data and variables during program execution.

1. Arithmetic Operators

Used to perform basic mathematical operations.


int a = 10, b = 5;
System.out.println(a + b); // 15
System.out.println(a - b); // 5
System.out.println(a * b); // 50
System.out.println(a / b); // 2
System.out.println(a % b); // 0

2. Relational Operators

Used to compare two values and return true or false.


int x = 10, y = 20;
System.out.println(x == y); // false
System.out.println(x != y); // true
System.out.println(x < y);  // true

3. Logical Operators

Used for combining multiple conditions.


int marks = 85;
System.out.println(marks > 50 && marks < 100); // true
System.out.println(marks > 90 || marks == 85); // true
System.out.println(!(marks < 50)); // true

4. Assignment Operators

Used to assign values to variables.


int num = 10;
num += 5;  // num = 15
num *= 2;  // num = 30

5. Ternary Operator

Short way of writing an if-else statement.


int age = 18;
String result = (age >= 18) ? "Adult" : "Minor";
System.out.println(result);

Understanding operators helps you write efficient and expressive Java code!
